## 文章排版

博客支持丰富的排版功能，让你的文章更加美观易读。以下是一些常用的排版元素：

### 文本样式

你可以使用各种文本样式来突出重点：

- **粗体文本** - 用于强调重要内容
- *斜体文本* - 用于表示特殊含义
- ~~删除线~~ - 用于表示过时内容
- `行内代码` - 用于表示代码片段

### 列表演示

有序列表：
1. 第一步：准备工作
2. 第二步：开始编写
3. 第三步：发布文章

无序列表：
- 支持深色模式
- 响应式设计
- 流畅的动画效果

### 引用块

> 这是一段引用文本。可以用来引用名人名言或者重要的观点。
>
> 支持多行引用内容。

## 代码展示

博客支持语法高亮，让代码更易读：

```javascript
// JavaScript 示例
function greet(name) {
    return `你好，${name}！`;
}

console.log(greet('世界'));
```

```css
/* CSS 示例 */
.button {
    background: #07C160;
    color: white;
    padding: 10px 20px;
    border-radius: 4px;
    transition: all 0.3s ease;
}
```
